# Paytm Application

## Backend -

> _Using express for routing_

> _Using mongoDB as Database_

> _Using cors for allowing frontend to connect to backend_ 

> _Using zod for input check_

> _Using jsonwebtoken for creating and authenticating the users_

> _Using mongoose to connect and perform CRUD on NoSQL database_ 

### Database -
- User 
- Account 
- Transactions

### Route -
- signup  (to sign up new user)
- signin  (to log in existing user)
- update  (to update information of user)
- search  (to search registered users)
- send    (to sent money to other users)
- my-data (to get all detail of the user)

### Upcoming features -

- Pagination for searching route
- Log-out route 
- not allowing user to sent money to himself/herself
- password hashing
- adding list of the transactions happened on specific account

## Frontend 

### Pages -
- signup 
    > Registration of new user by unique Username and Password and providing token for further passage

    > Bridge to signin page to allow registered user to get to signin page directly
- signin
    > Authentication of user and providing token for further passage
- content-page
    > Allow user to transfer money other users just by receiver username and amount to be send

    > Search and Filter registered users
- my-account
    > This page consist of : Username - Current balance - Transaction History
