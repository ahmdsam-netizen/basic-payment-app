# Paytm Application

## Backend -

- _Using express for routing_
- _Using mongoDB as Database_
- _Using cors for allowing frontend to connect to backend_
- _Using zod for input check_ 
- _Using jsonwebtoken for creating and authenticating the users_ 
- _Using mongoose to connect and perform CRUD on NoSQL database_ 

### Route -
- signup (to sign up new user)
- signin (to log in existing user)
- update (to update information of user)
- search (to search registered users)
- send   (to sent money to other users)

### Upcoming features -

- Pagination for searching route
- Log-out route 
- not allowing user to sent money to himself/herself
- password hashing
- guarding route from direct access
- adding list of the transactions happened on specific account
